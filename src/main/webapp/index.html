PTC test Sample Liberty Application Connects to IBM MQ on Cloud

Run the sample application via Curl command with the deployed application URL from your EASe Java instance:

curl -v -k -X POST -d 'msg=testing messages with IBM MQ on Cloud' https://your_deployed_sample_application_network_route_URL/libertymq/api/enqueue

To find the deploymed URL, check deployed job log

For example:

curl -v -k -X POST -d 'msg=testing messages with IBM MQ on Cloud' https://mei-easej-mq-academic-way-8529.staging.us-east.c2.appflow.dev.ibmappdomain.cloud/libertymq/api/enqueue

See the output in application runtime log from EASe Java UI:

[SystemO] Message enqueued.

[SystemO] MDB received: testing messages with IBM MQ on Cloud
